# 🚀 NASA Artemis Trajectory Analysis

This project explores publicly available NASA Artemis II mission trajectory data using Python to analyze spacecraft motion and visualize trajectory patterns.

---

## 📌 Overview

The dataset consists of raw spacecraft telemetry in ASCII (`.asc`) format, including:

- Timestamped observations  
- Position coordinates: `x, y, z`  
- Velocity components: `vx, vy, vz`  

The goal of this project is to transform unstructured telemetry data into a structured format and derive meaningful insights from it.

---

## 🎯 Objectives

- Parse raw `.asc` telemetry data  
- Convert data into a structured DataFrame  
- Perform feature engineering  
- Analyze spacecraft motion  
- Visualize trajectory and trends  

---

## ⚙️ Key Features

- Data parsing from raw ASCII telemetry  
- Feature engineering:
  - Speed calculation  
  - Distance from Earth  
- Time-series analysis  
- 2D trajectory visualization  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Jupyter Notebook  

---

## 📊 Sample Analysis

- Computed spacecraft speed using velocity vectors  
- Calculated distance from Earth using 3D coordinates  
- Visualized trajectory in X-Y plane
